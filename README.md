# react-native-calculator
A react native project to build a simple calculator.

The component we used in this project are as follows:
  1. Text
  2. View
  3. StyleSheet
  4. TouchableOpacity
  5. ScrollView
  6. SafeAreaView
  7. TextInput
  
  <i>(Note : The Values of currency are static and placed into App.js file)</i>
